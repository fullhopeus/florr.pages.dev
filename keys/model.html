Key's error code
If the error code is "Online key", the windows activate immediately.
If the key's error code is "0xC004C008", the windows will generate an information ID, and the IID is used for auto-call system. Mostly, keys are always this status.
If the key's error code is "0xC004C020", the windows will generate an information ID too, but the IID is used for artificial system. See What is IID? for more details.
If the key's error code is "0xC004C060", it means the key has been expired.
If the key's error code is "blocked", it means that the key has been blocked and it cannot be used for activate.
If the key's error code is "0xC004C003", it means the key has been expired, too.

What is IID?
IID is an ID that generated by Windows. It will change if the configuration of the computer changes.
IID is used for 0xC004C008 keys and 0xC004C020 keys. Each IID has a CID.
